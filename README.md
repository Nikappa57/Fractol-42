## Fractals
- [x] Mandelbrot
- [x] Julia
- [x] Newton

## Install
```
git clone https://github.com/Nikappa57/Fractol-42.git
cd Fractol-42
make
```

## Usage
```
./fractol [mandelbrot | newton] <max iteration>
./fractol julia <max iteration> (real) (imaginary)
```
